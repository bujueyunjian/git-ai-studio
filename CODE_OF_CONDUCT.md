# Code of Conduct

This project adopts the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

We expect everyone participating in our community — through issues, pull requests, discussions, code reviews, or any other interaction — to read and abide by it.

## Reporting

If you witness or experience a violation, please report it by opening a confidential issue tagged `conduct`, or by emailing the maintainers (contact will be listed in `MAINTAINERS.md` once published). Reports are reviewed promptly and confidentially.

## Scope

This Code of Conduct applies to all project spaces, online and offline, including issues, pull requests, code reviews, commit messages, project social channels, and in-person events.
